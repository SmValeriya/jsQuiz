## Викоростанні технології

- Pug
- POSTCSS
- Yarn
- Gulp
- Webpack
- Browsersync

## Перше використаня

```sh
git clone git@github.com:SmValeriya/gulp-template.git
cd gulp-template
yarn install
yarn husky install
```

## Скрипти

1. Для розробки
```sh
yarn gulp
```
2. Для продакшена
```sh
yarn buld
```
